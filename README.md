# ブロッククラフト - 2D設計図メーカー

マインクラフトのブロックを使って、画像を2D設計図に変換するツールです。

## 🚀 使い方 (開発モード)

パソコンを再起動した後などに、このプロジェクトをもう一度動かす手順です。

1.  **ターミナルを開く**:
    VSCodeなどのエディタでターミナルを開きます (`Ctrl + @` または `Cmd + J`)。

2.  **コマンドを実行**:
    以下のコマンドを入力してエンターキーを押します。

    ```bash
    npm run dev
    ```

3.  **ブラウザで開く**:
    `http://localhost:3000` にアクセスすると、ツールが使えます！
